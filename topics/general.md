# General Software Development

Beyond domain-specific research software, I develop tools and utilities that streamline workflows, improve productivity, and support the broader software development community.

## Development Focus

### Developer Tools
- Scripts and utilities for automation
- Configuration management for development environments
- Tools for software reproducibility

### Scientific Computing Infrastructure
- Build systems for complex software stacks
- Continuous integration for scientific code
- Containerization for portable deployments

### Utilities and Libraries
- Reusable components for common tasks
- Testing and validation tools
- Documentation and examples

## Technical Stack

My general software development leverages:
- **Version Control**: Git for collaborative development
- **CI/CD**: GitHub Actions for automated testing and deployment
- **Containerization**: Docker and Apptainer for reproducible environments
- **Package Management**: Conda/Mamba, Spack for dependency management
- **Build Systems**: CMake for complex multi-language projects
- **Scripting**: Bash and Python for automation

## Best Practices

I emphasize:
- Code quality through testing and review
- Documentation for long-term maintainability
- Reproducible development environments
- Modular design for reusability
- Open source licensing when appropriate
